# Apostila Pilhas e Filas - Entrega

Projeto Java com os exercicios do capitulo 8 e dois desafios do capitulo 9 da Apostila_Pilhas_Filas_v2.

## Conteudo implementado

### Capitulo 8

1. Inversao de string com pilha
2. Avaliador de expressao pos-fixa (com operador %)
3. Fila usando duas pilhas
4. Comparacao empirica com System.nanoTime()
5. Rate limiter com janela deslizante

### Capitulo 9 (2 desafios)

1. Pilha com minimo em O(1)
2. Buffer circular de logs

## Testes

Executar no diretorio do projeto:

mvn test
